# MNEG-Wheat-Disease-classification

# 1.研究背景于模型定位
小麦是全球种植面积最广、消费量最大的粮食作物之一，占世界谷物总产量的三分之一以上。但小麦叶片病害（如锈病、白粉病、叶枯病等）的频繁发生，严重影响小麦的产量与质量。  
传统人工识别小麦叶片病害存在耗时长、依赖专业经验、大规模田间检测效率低等问题。为此，我们提出MNEG 模型，通过融合阶梯式双注意力机制，增强对病害特征的细粒度提取能力，同时抑制复杂背景干扰，为小麦病害检测提供精准、高效的深度学习解决方案
